ML‑сервис для анализа текста
Описание
Это сервис для работы с ML‑моделями анализа текста. Система позволяет:

управлять пользователями (включая администраторов);

отслеживать баланс кредитов для выполнения анализа;

запускать задачи анализа текста через ML‑модели;

хранить историю выполненных операций.
